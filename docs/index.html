About Sightline

Sightline is a suite of privacy-aware media tools. Built to make professional-grade redaction and processing accessible to everyone, it combines powerful automation with intuitive controls.

Whether you're a journalist protecting sources, a researcher anonymizing data, or a creator preparing content, Sightline ensures your work remains private and secure by performing all operations locally on your device.

Core Technologies

Sightline is built on the shoulders of giants. It leverages state-of-the-art open source libraries including:

  • deface - Face detection and blurring
  • WhisperX - Fast speech-to-text and alignment
  • pyannote.audio - Speaker diarization
  • tkinterdnd2 - Drag and drop support
  • OpenCV - Computer vision
  • NumPy - Numerical computing
  • Pillow - Image processing
