Horizon motif

Use the horizon line sparingly as a separator or hero motif. It should feel like a single beam of clarity cutting through darkness.

Typography

Primary Typeface
Sightline Interface
SF Pro / system UI, 400–600 weight, slightly expanded letter-spacing.
Logo & Brand Name
Set in all caps with letter-spacing: 0.3em. Use only for the main brand wordmark.
Headlines
1.3–1.7rem, weight 500–600, with 0.02–0.04em letter-spacing. Avoid overly heavy 700+ weights.
Body Text
0.9–1rem, weight 400–500, line-height ~1.55. Color #8EA4C7 for muted copy to reduce glare.

UI Components

Components should feel minimal and cinematic. Motion is subtle: tiny lifts, soft glows, and horizon accents rather than large color shifts.

Primary buttons are reserved for the main workflow action (e.g., “Select image”, “Process frame”). All other actions should be secondary or text links.

Focus & Hover
Use a faint glow ring and tiny lift.
outline: 2px solid rgba(0,166,255,0.5);
transform: translateY(-1px);
Layout Rhythm
Prefer centered layouts with generous padding. Avoid dense clusters of UI elements; Sightline should feel calm and deliberate, not busy.
